# Mission Blueprint

## Mission name
Ferry landing page + launch one-pager

## Objective
Ship a single-page marketing site and a launch one-pager for "Ferry," a simple tool that lets someone send a file via a link that expires after a set time or number of downloads. Build to local/staging in test mode, validated by 2 user personas, with positioning and page copy consistent across both deliverables.

## Definition of done
- [ ] Winning concept selected and audited: positioning line, hero message, 3 value props, primary CTA
- [ ] Landing page built to local: single index.html, responsive, working click-through, signup wired in test mode (no real backend, no real email capture)
- [ ] Launch one-pager built: target audience, 3 channels, pre-launch checklist
- [ ] Cross-consistency audited: the page's claims, price framing, and CTA match the one-pager
- [ ] 2 persona tests run, findings synthesized, ship/iterate decision made
- [ ] Run report + retrospective delivered, lessons written to the ledger

## Constraints
- **Budget cap:** none (nothing is spent regardless; this is a staging-only demo)
- **Deadline:** none
- **Brand tokens:** to be created in run (Ferry is fictional; keep it minimal and clean)
- **Compliance / hard rules:** no em dashes anywhere; no real signup, payment, or email capture (test mode only); the product is fictional, so no claims that imply a real shipping service

## Autonomy (Dial 1: execution)
FULL

## External authority (Dial 2: blast radius)
Tier 1 (build to local/staging only; no external sends, posts, money, or signatures)

## Run budget (stop conditions)
- Max waves: 8
- Max total dispatches: 30
- Max iterate loops: 3

## Pre-supplied inputs (grounding, skips the research wave)
This is a demo run, so grounding is supplied here rather than via a researcher dispatch:
- **Market framing:** crowded space (WeTransfer, Firefox Send was discontinued, Smash, Send Anywhere). Ferry's wedge is the expiring link (time or download-count limit) for privacy-conscious one-off sends.
- **Audience:** freelancers and small studios sending client deliverables who want the link to not live forever.
- **Price framing (fictional):** free up to 2GB and 7-day expiry; "Pro" at a flat low monthly for larger files and custom expiry. Stated as framing only, no checkout.

## Notes for the orchestrator
Keep deliverables tight: one index.html, one one-pager. The point of this run is to prove the engine end to end (plan, parallel build, gates, persona fan-out, retrospect, ledger), not to ship a large site. Skip the research wave (grounding above). Run ledger + retrospective at the end.
